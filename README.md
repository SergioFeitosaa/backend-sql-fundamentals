# PostgreSQL CRUD Fundamentals

Repositório criado para fortalecer os fundamentos de banco de dados com foco em desenvolvimento Backend utilizando Java e Spring Boot.
Este projeto contém exercícios práticos de SQL utilizando PostgreSQL, abordando operações CRUD, criação de tabelas, manipulação de dados, 
filtros e consultas utilizadas no dia a dia de aplicações backend.

🚀 Tecnologias Utilizadas
- PostgreSQL
- SQL
- Git
- GitHub


🧠 Objetivo do Repositório

O principal objetivo deste repositório é documentar minha evolução nos estudos de banco de dados e consolidar fundamentos essenciais para o desenvolvimento backend.

Os estudos serão evoluídos gradualmente com conceitos mais avançados de SQL e futura integração com aplicações Java utilizando Spring Boot e Spring Data JPA.
